Shopify Summer '26 Editions: 明日6月17日に150+アップデートを発表予定 — Checkout Components GA・ネイティブAIマーチャンダイジング・Expiring OAuthトークン対応

shopify
🔗 novadata.io

Shopify Summer '26 Editions が明日6月17日のライブストリームで公開される。150件以上のアップデートのうち約65件が開発者向け変更で、Checkout Components の Plus 向け GA・ネイティブ AI マーチャンダイジング・OAuth 2.0 準拠の Expiring Token が含まれる。Shopify Functions への移行期限（6月30日）も迫っており、Scripts 利用中のアプリは早急な対応が必要。6月5日には Native A/B Testing（Rollouts）も全ストアに GA 公開済みで、テーマ・チェックアウト変更を安全に段階的に展開できる。

GraphQL Admin API 2026-07: ローカル通貨ギフトカードの発行・換算利用に対応 — issuanceCurrency フィールドと換算利用制御フラグを追加

shopify
🔗 shopify.dev

API バージョン 2026-07 より、GraphQL Admin API でローカル通貨のギフトカード発行が可能になった。issuanceCurrency フィールドで発行通貨を指定でき、null の場合はショップ通貨で発行される。購入時は指定通貨でのみ購入可能となり、他通貨での換算利用も設定で選択可能。旧来の initialValue ミューテーションは非推奨となり、新しい gift card product ミューテーションへの移行が推奨される。

security
🔗 SecurityWeek

Deno Deploy Classic: 7月20日でサービス終了 — dash.deno.com から console.deno.com への移行必須、プロジェクトの自動引き継ぎなし

dev
🔗 Deno Docs

Deno Deploy Classic（dash.deno.com）が2026年7月20日をもってサービスを終了する。新規組織のオンボードはすでに停止されており、既存プロジェクトは新プラットフォーム（console.deno.com）へ手動で移行する必要がある。旧プラットフォームのプロジェクトは新プラットフォームへ自動引き継ぎされず、標準ライブラリの旧 serve() 関数から組み込みの Deno.serve() API への書き換えも必要となる。
