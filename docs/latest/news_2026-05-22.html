Shopify CLI — `shopify app release --force` フラグが非推奨に、CI/CD パイプライン破壊のリスクに注意

shopify
🔗 Shopify Developer Changelog

Shopify CLI の `shopify app deploy` および `shopify app release` コマンドから `--force` フラグが非推奨（deprecated）となり、近日中に削除される予定。CI/CD パイプラインで `--force` フラグを使用している場合、削除後はビルドが失敗するため早急な対応が必要。Shopify は通常のデプロイフローへの移行を推奨しており、`--force` なしのコマンドで対応できるよう CI/CD スクリプトの確認と更新が求められる。

CVE-2026-20182 — Cisco Catalyst SD-WAN に CVSS 10.0 の認証バイパス、APT グループ UAT-8616 による実攻撃を CISA KEV に追加

security
🔗 The Hacker News

Cisco Catalyst SD-WAN Controller に CVSS スコア 10.0 の重大な認証バイパス脆弱性（CVE-2026-20182）が発見され、CISA の Known Exploited Vulnerabilities（KEV）カタログに追加された。vHub デバイスを偽称することで証明書検証をスキップしながら認証済みとしてマーキングされるバグを悪用して管理者権限を取得でき、NETCONF 経由で SD-WAN ファブリック全体の設定変更が可能になる。Cisco Talos は APT グループ UAT-8616 による少なくとも 2023 年以来の継続的な悪用を追跡しており、連邦民間行政府（FCEB）機関は 2026/05/17 が修正期限。2026 年に入り Cisco SD-WAN で実攻撃が確認されたゼロデイとして 6 件目。

DirtyDecrypt（CVE-2026-31635）の PoC が公開 — Linux カーネル RxGK モジュールの CoW 欠如で root 昇格、Fedora・Arch・openSUSE に影響

security
🔗 The Hacker News

Linux カーネルの RxGK サブシステム（AFS クライアント向け GSS-API 認証層）の `rxgk_decrypt_skb()` 関数に Copy-on-Write ガードが欠落しているため、非特権ユーザーがページキャッシュを書き換えて `/etc/shadow` や SUID バイナリを上書きし root 権限を取得できる脆弱性（CVE-2026-31635、通称 DirtyDecrypt）の PoC が 2026/05/18 に Zellic・V12 チームにより公開された。CONFIG_RXGK が有効な Fedora・Arch Linux・openSUSE Tumbleweed が対象で、Debian Stable・RHEL・Ubuntu LTS はデフォルト非影響。アップストリームパッチは 2026/04/25 に統合済みで、各ディストリビューションのカーネル更新を速やかに適用すること。
