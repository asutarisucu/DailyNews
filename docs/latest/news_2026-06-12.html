npm v12（2026年7月リリース予定）破壊的変更を予告 — 依存スクリプト実行・Git 依存・リモート URL をデフォルトでブロック

dev
🔗 GitHub Changelog

GitHub は npm v12（7月リリース予定）の3つの破壊的変更を発表した。①npm install 時に依存パッケージの preinstall/install/postinstall スクリプトを明示的な許可なしに実行しない、②Git 依存（直接・推移的）を --allow-git なしで解決しない、③リモート URL 依存（https tarball 等）を --allow-remote なしで解決しない。Red Hat npm スコープ侵害や Phantom Gyp 攻撃など最近のサプライチェーン攻撃への対策として設計されており、現行 npm 11.16.0 以降で警告として表示されているため早めの確認が推奨される。
