🛒 Shopify 開発者情報

Next Generation Events が Developer Preview に登場 — フィールドレベルトリガーとカスタム GraphQL ペイロードで Webhook を刷新

shopify
🔗 Shopify Developer Changelog

Shopify の次世代イベントシステムが Developer Preview として公開された。従来の Webhook と異なり、フィールドレベルのトリガーを設定することで price 変更時のみ配信するといった細粒度な制御が可能になる。ペイロードは標準 Admin GraphQL クエリで定義でき、不要なフィールドの過剰取得を排除。また各デリバリには _changed フィールドが付与され、どのフィールドが変更をトリガーしたか明示される。現在 Product・Customer トピックが利用可能で、unstable API バージョンで試せる。

Shopify Next-gen Dev Platform が全アプリ開発者に GA — Remix/Vite ベースの開発体験が正式解放

shopify
🔗 Shopify Developer Changelog

Remix・Vite を基盤とした次世代 Shopify 開発プラットフォームが全アプリ開発者に正式公開（GA）された。HMR（Hot Module Replacement）によるリアルタイムプレビュー、TypeScript ファーストの開発体験、最新 Shopify CLI との深い統合が提供される。従来の Node.js テンプレートから移行するガイドも整備されており、新規アプリは次世代プラットフォームをデフォルトで採用する形になる。

OpenAI「Guaranteed Capacity」開始 — 1〜3年コミットでコンピュートを長期予約、割引率は期間に応じて増加

ai
🔗 CNBC

OpenAI がエンタープライズ向けに「Guaranteed Capacity」プログラムを発表した。1年・2年・3年の長期コミット契約で OpenAI のコンピュートリソースを確保でき、期間が長いほど割引率が高くなる仕組み。Sam Altman CEO は「顧客からキャパシティの確実性を求める声が増えている」とし、モデル性能向上に伴いコンピュートが希少資源になるとの見通しを示した。Microsoft Azure に加えた追加ホスティングパートナーの活用も背景にある。

Claude Managed Agents — 自己ホスト型サンドボックスとプライベート MCP トンネルを追加、エンタープライズセキュリティを強化

ai
🔗 Anthropic / Claude Blog

Anthropic が Claude Managed Agents に 2 つの新機能を追加した。「自己ホスト型サンドボックス」では、エージェントの tool call を Cloudflare・Daytona・Modal・Vercel などの自社インフラ上で実行でき、プライベート API やカスタマーデータをクラウド外に漏らさずに済む。「プライベート MCP トンネル」では、エンタープライズ境界内でのみアクセス可能なプライベート MCP サーバーへ接続可能になる。認証情報はネットワーク境界でサンドボックスに注入されるため、エージェントコンテキストには直接触れない。

Google Gemini 3.5 Flash 登場 — フロンティアモデルの約 1/3 の価格、AI Ultra プランも 20% 値下げで $200/月に

ai
🔗 CNBC

Google が Google I/O 2026 で Gemini 3.5 Flash を発表した。軽量モデルながら最先端の能力を備え、価格は同等フロンティアモデルの約 1/3〜1/2 という競争力のある設定。同時に最上位プラン「AI Ultra」を月額 $250 から $200 に値下げし、入門レベルの AI Ultra プランを $100/月で新設した。Gemini Spark（汎用 AI エージェント）は来週より AI Ultra 加入者から優先提供予定。

Pwn2Own Berlin 2026 閉幕 — 47 件のゼロデイで総額 $129 万、DEVCORE が Master of Pwn 獲得

security
🔗 BleepingComputer

5/14〜16 に開催された Pwn2Own Berlin 2026 が終了し、研究者たちは 47 件のゼロデイを実証して合計 $1,298,250 を獲得した。DEVCORE チームが Microsoft SharePoint・Exchange・Edge・Windows をすべてハックして 50.5 ポイントで総合優勝。AI プラットフォームカテゴリでは LiteLLM・OpenAI Codex も標的となった。ベンダーはゼロデイ開示から 90 日以内にパッチをリリースする義務がある。Windows 11 や Microsoft Exchange を含む多数の完全パッチ済み製品が突破された。

【CISA KEV】5/20 に 7 件追加 — Microsoft Defender の特権昇格・DoS を含む、連邦機関に対応期限

security
🔗 CISA

CISA が 2026/05/20 に 7 件の脆弱性を Known Exploited Vulnerabilities (KEV) カタログに追加した。主なものとして CVE-2026-41091（Microsoft Defender 特権昇格）と CVE-2026-45498（Microsoft Defender DoS）が含まれ、いずれも実際の攻撃での悪用が確認されている。BOD 22-01 に基づき FCEB（連邦民間行政府）機関は指定期限までの修正が義務付けられており、民間組織にも速やかな対応が推奨されている。

Google × Blackstone — $50 億規模の TPU クラウド JV を設立、2027 年までに 500MW の AI インフラを稼働

cloud
🔗 CNBC

Google と Blackstone が合弁会社を設立し、Blackstone が初期株式資本として $50 億を投資すると発表した。Google 独自のチップ「TPU（Tensor Processing Unit）」を商業化し、Nvidia GPU 依存の市場に対抗する計算インフラを提供することが目的。2027 年までに最初の 500MW を稼働させる計画で、CEO には Google の元チーフプログラムオフィサー Benjamin Treynor Sloss が就任する。HPC・AI ワークロード向けのコンピュートアズアサービスを中心に展開予定。

Vercel Sandbox で Claude Managed Agents の tool call を実行可能に — Firecracker microVM でエンタープライズ分離を実現

cloud
🔗 Vercel Changelog

Vercel が Claude Managed Agents との統合を発表し、エージェントの tool call を Vercel Sandbox 上で実行できるようになった。各エージェントセッションは独立した Firecracker microVM 内で動作し、VPC ピアリングによってプライベート API やバックエンドサービスにアクセスできる。認証情報はサンドボックス外のネットワーク境界で注入されるため、エージェントコンテキストには秘密情報が直接渡らない。Vercel の既存インフラを利用するため、ミリ秒単位の起動時間とエンタープライズグレードの可用性を備える。

Vercel — Flat Rate CDN を Pro チーム向けに限定ベータ開始、Edge リクエスト込みの定額制でトラフィックスパイクを無課金化

cloud
🔗 Vercel Changelog

Vercel が Pro チーム向けに Flat Rate CDN の限定ベータを開始した。従来の従量課金型 CDN に代わり、毎月定額料金で Edge リクエスト・Fast Data Transfer を含む CDN 利用がカバーされ、トラフィックスパイク時の想定外請求がなくなる。WAF で拒否・チャレンジ・レート制限されたトラフィックは CDN 課金の対象外になるファイアウォール請求救済措置も同時に適用される。高トラフィックのプロダクションアプリを運営するチームには大幅なコスト削減が期待できる。

🛠️ 開発ツール・OSS

Vercel — モノレポ向け統合 GitHub コミットステータスを提供開始、PR ごとのステータスを 1 つに集約してブランチ保護を簡素化

dev
🔗 Vercel Changelog

多数のプロジェクトを含むモノレポでは PR ごとに大量のコミットステータスが発生し、GitHub のブランチ保護ルール設定が煩雑になる課題があった。Vercel の新機能「Consolidated Commit Status」を有効化すると、すべての Vercel プロジェクトのステータスが PR 上で 1 つに集約される。ブランチ保護ルールは一度だけ設定すればよく、マージに必要な個別プロジェクトはプロジェクト設定で管理できる。オプトイン形式で既存プロジェクトへの影響はない。

Vercel AI Gateway — Gemini 3.5 Flash を追加、改善されたコーディング能力と並列エージェント実行ループをサポート

dev
🔗 Releasebot / Vercel

Vercel AI Gateway に Google の Gemini 3.5 Flash が追加された。改善されたコーディング能力と並列エージェント実行ループをサポートし、コスト効率の高いモデルとして AI SDK から直接利用できる。AI Gateway はコスト・TTFT（Time to First Token）・TPS（Tokens per Second）でプロバイダーをランキングする機能も備えており、ユーザーは最適なモデルを簡単に比較・選択できる。既存のプロバイダーランキング機能と組み合わせることで、Gemini 3.5 Flash を候補に加えた自動フォールバック設定も可能。

Meta が全社員の 10% にあたる 8,000 人を削減 — AI 再構築の一環、年内にさらに 2 ラウンドの削減予定

other
🔗 The Next Web
