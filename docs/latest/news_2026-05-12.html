Shopify CLI の --force フラグが廃止予定 — --allow-updates / --allow-deletes への移行が必要

shopify
🔗 shopify.dev changelog

Shopify CLI の app release --force フラグが廃止予定となりました。このフラグはエクステンションの削除を含む全確認プロンプトをスキップするため、誤ってアプリ拡張を削除してしまうリスクがありました。代わりに --allow-updates および --allow-deletes フラグを利用することで、CI/CD パイプラインの安全な無人実行が可能になります。すでに --force を使用しているスクリプトは早急に更新してください。
