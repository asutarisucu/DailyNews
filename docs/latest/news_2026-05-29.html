Shopify CLI 4.0 リリース — SemVer 移行・自動アップデート・非推奨フラグの完全削除

shopify
🔗 Shopify Developer Changelog

Shopify CLI がメジャーバージョン 4.0 に更新され、バージョニングをロマンティック方式から Semantic Versioning（MAJOR.MINOR.PATCH）に変更。インストール時に使用したパッケージマネージャーで自動アップグレードが行われるようになり、CI 環境・ローカルインストール・メジャーバージョンアップ時はスキップされる。また --force フラグが完全に削除され、代わりに --allow-updates と --allow-deletes フラグでデプロイ操作の粒度を細かく制御できるようになった。自動アップグレードを無効にするには shopify config autoupgrade off を実行する。

GraphQL Admin API の Order オブジェクトに cartToken フィールドを追加（API 2026-07）

shopify
🔗 Shopify Developer Changelog

GraphQL Admin API バージョン 2026-07 から、Order オブジェクトに cartToken フィールドが追加された。これにより GraphQL でも REST Admin API の cart_token フィールドと同等の情報を取得でき、注文を作成したカートとの紐づけが容易になる。Headless コマースやカスタム注文管理フローで、カートから注文への追跡をシンプルに実装できる。
