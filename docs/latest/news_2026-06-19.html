🛒 Shopify 開発者情報

Shopify Scripts 移行支援レポート公開 — 既存スクリプトのFunctions移行可否を自動診断、6月30日廃止まで残り11日

shopify
🔗 Shopify Changelog

Shopifyが「Scripts Customizations Report」をリリースした。管理画面から現在稼働中のScriptsを一覧し、割引・配送ロジック・支払いカスタマイズのそれぞれについてShopify Functionsへの移行可能性を自動診断できる。6月30日の完全廃止まで残り11日となっており、7月1日以降はスクリプトが即停止するため、このレポートを使って移行漏れを最終確認することが強く推奨されている。Plus専用カスタムアプリへの移行か、公開アプリの採用かも同レポート上で判別可能。

Admin GraphQL API 2026-07 新フィールド群: Channel Markets・Inventory Transfer Metafields・LineItem.weight が追加 — DraftOrderLineItem.grams は同バージョンで削除

shopify
🔗 shopify.dev

Admin GraphQL API 2026-07で複数のフィールドが追加・削除される。新機能としてChannel Marketsにより販売チャネルごとのマーケット作成・更新・照会が可能となり、チャネル別価格設定や通貨管理が実現する。Inventory TransferへのMetafield定義・設定が可能になり、転送固有のカスタムデータを一括で書き込めるようになった。また LineItem.weight フィールドが公開GraphQL APIに追加される一方、8年以上前に非推奨化された DraftOrderLineItem.grams フィールドは2026-07で完全削除となるため、weight フィールドへの移行対応が必須となる。

shopify
🔗 Shopify Changelog

Cisco ISE 二段階攻撃チェーン: CVE-2026-20190（認証なし資格情報漏洩）→ CVE-2026-20181（CVSS 9.1 RCE）— ISE 3.5 Patch 4は8月まで未提供

security
🔗 SecurityWeek

Cisco Identity Services Engine（ISE）に深刻な二段階攻撃が成立する脆弱性が公表された。まずCVE-2026-20190（不適切な認可チェック）を使い、認証なしでハッシュ化された管理者資格情報を奪取する。次にCVE-2026-20181（CVSS 9.1、コマンドインジェクション）でその資格情報を使って任意コード実行からroot権限昇格まで完遂できる。ISE 3.3 Patch 11とISE 3.4 Patch 6には修正済みだが、ISE 3.5のパッチは2026年8月まで提供されないため、3.5系を使用する組織は設定回避策の適用が急務となる。

security
🔗 CISA

Microsoftが "RoguePlanet" と命名したMicrosoft Defenderのゼロデイ脆弱性CVE-2026-50656（CVSS 7.8）が明らかになった。権限昇格を許す欠陥であり、Microsoftはパッチを準備中と発表しているが提供時期は未確定だ。この脆弱性はCISAのアドバイザリでも取り上げられており、パッチ適用までの間は最小権限原則の徹底やDefenderの監査ログ強化による異常検知が一時回避策として推奨されている。特権昇格を伴う侵害の連鎖に利用される可能性があるため、早急な対応計画の策定を推奨。
