shopify
🔗 Shopify Developer Changelog

本日2026年6月30日をもって、Shopify Scriptsが完全に実行停止となった。4月15日以降すでに編集・公開は不可だったが、今日からスクリプト自体の実行も完全に停止。カスタム割引・配送・支払いロジックをShopify Scriptsで実装していたPlus加盟店で、Shopify Functions または公開Appへの移行が未完了の場合、チェックアウト時のカスタムロジックが今日から一切動作しなくなる。2回延期された末の最終廃止日であり、Shopifyは再延期を行わないことを確認している。

Deno 2.9 リリース — `deno desktop` でWebアプリをネイティブデスクトップバイナリに変換、コールドスタートも17msに半減

dev
🔗 Deno Blog

Deno 2.9が6月25日にリリース。目玉機能「deno desktop」はNext.js・Astro・Fresh・SvelteKit等を自動検出し、WebViewベースのUIとDenoバックエンドを1バイナリに統合したネイティブアプリ（macOS .dmg、Windows .msi、Linux .AppImage等）を生成する。Electron不要でボイラープレートなし。コールドスタートもv2.8の34msから17msへ半減した。さらに `deno install` がnpm/pnpm/yarn/Bunのロックファイルを直接読み込めるようになり、既存プロジェクトへの採用障壁が大幅に低下している。
