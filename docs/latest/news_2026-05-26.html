shopify
🔗 Shopify Developer Changelog

security
🔗 The Hacker News

米国司法省とカナダ当局が合同捜査の末、KimWolf DDoS ボットネットの運営者とされるカナダ・オタワ在住の Jacob Butler（23 歳、ハンドル「Dort」）を逮捕・起訴した。KimWolf は AISURU の亜種で Android デバイスの公開 ADB サービスを悪用して約 200 万台に感染し、DDoS-as-a-Service モデルで他の犯罪者に攻撃インフラを販売していた。記録された攻撃は約 30 Tbps と、これまでで最大規模の DDoS 攻撃量となった。逮捕に合わせて 45 の DDoS-for-Hire プラットフォームを支援するサービスへの差押令状が執行され、インフラの解体が進められている。有罪の場合は最大 10 年の禁固刑。

Deno 2.8 リリース — deno audit fix・deno why など 6 つの新サブコマンド、npm インストールが 3.66 倍高速化、npm: プレフィックス不要に

dev
🔗 Deno Blog

Deno 2.8 が 5 月 22 日にリリースされ、マイナーリリースとして過去最大規模の機能追加となった。新たに deno audit fix・deno why・deno transpile・deno pack・deno bump-version・deno ci の 6 つのサブコマンドが追加された。deno audit fix は影響を受けるパッケージを最近のパッチバージョンに自動アップグレードし、deno why は npm explain 相当の依存関係説明機能を提供する。CLI での npm: プレフィックスが不要になり、deno add や deno install でプレフィックスなしのパッケージ名が npm パッケージとしてデフォルト処理される。コールドキャッシュの npm インストール速度が 2.7 比で 3.66 倍に改善されており（95% 信頼区間 3.53〜3.75 倍）、Chrome DevTools でのネットワークデバッグも新たにサポートされた。
