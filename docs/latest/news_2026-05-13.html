Storefront API: 購入者の所在地で購入不可の商品をカートに入れると警告を返すように（2026-07版～）

shopify
🔗 shopify.dev changelog

Storefront API のバージョン 2026-07 以降、購入者の所在地で販売不可の商品をカートに追加しようとすると警告（warning）が返されるようになります。これにより越境EC対応アプリはユーザーに対してより適切なエラーメッセージや代替案を提示できます。既存実装でカートのレスポンスを処理している箇所は warnings フィールドへの対応を事前に確認しておくことが推奨されます。

GraphQL Admin API: Order オブジェクトに cartToken フィールドを追加 — 注文元カートの追跡が可能に

shopify
🔗 shopify.dev changelog

GraphQL Admin API の Order オブジェクトに cartToken フィールドが追加されました。これにより、注文を生成したカートのトークンをクエリで直接取得できるようになり、カート放棄の追跡やアナリティクス連携が容易になります。Storefront API のカートと Admin API の注文を紐づける処理を実装しているアプリで特に有用です。
