Shopify Scripts 廃止まで残り20日 — 4月15日に編集停止済み、本番バグ修正不可、今すぐ Functions 移行を

shopify
🔗 Shopify Developer Changelog

Shopify Scripts は6月30日（JST）に完全停止し、すべてのカスタム割引・配送・決済スクリプトが実行されなくなる。4月15日にはスクリプトの編集・新規公開が停止されており、本番で不具合が起きても Script Editor での修正は不可能。管理画面「アプリ > Script Editor」の「Shopify Scripts を置き換える」バナーから移行レポートにアクセスし、対応する Shopify Functions アプリへ切り替える必要がある。Scripts と Functions は同一ストアで同時稼働可能なため、段階的な移行が推奨される。

LiteLLM CVE-2026-42271（CVSS 8.7）野外悪用確認・CISA KEV 追加 — MCP エンドポイント経由コマンドインジェクション + BadHost バイパスで未認証 RCE 成立

security
🔗 The Hacker News

CISA が LiteLLM のコマンドインジェクション脆弱性 CVE-2026-42271（CVSS 8.7）を既知悪用脆弱性カタログ（KEV）に追加した。MCP サーバーの接続テスト用エンドポイント（POST /mcp-rest/test/connection）がリクエストボディで渡された stdio 設定のコマンドをそのままサブプロセスとして実行する欠陥で、Starlette の「BadHost」ホストヘッダー検証バイパス CVE-2026-48710 と連鎖することで認証なしの RCE が成立する。影響バージョンは 1.74.2〜1.83.6 で、v1.83.7 で修正済み。AIゲートウェイ経由でモデルプロバイダーのAPIキーや CI/CD シークレットが奪われる恐れがあるため、即時アップデートが強く推奨される。

Red Hat npm @redhat-cloud-services 32 パッケージ侵害 — GitHub Actions OIDC 経由 CI/CD ハイジャック、週 11.7 万ダウンロードに資格情報窃取ワーム混入

security
🔗 Palo Alto Networks Unit 42

6月1日、Red Hat の公式 npm スコープ @redhat-cloud-services から公開された 96 バージョン／32 パッケージに資格情報窃取ワームが混入していることが確認された。パッケージは GitHub Actions の OIDC トークンを悪用して CI/CD パイプラインから直接 npm に公開されており、npm トークンの漏洩ではなくパイプライン自体のハイジャックが原因。各パッケージには package.json の preinstall スクリプトが仕込まれており、npm install 実行時に自動的にペイロードが起動する。クラウドプロバイダー・CI/CD ツール・開発者ツールを横断して広範な資格情報を収集するマルウェアで、影響を受けるパッケージを利用していた環境では全シークレットのローテーションが必要。

npm 新攻撃手法「Phantom Gyp」 — 6月3日に57パッケージ追加侵害、署名済みアテステーションもパイプラインハイジャックには無効

security
🔗 TechTimes

Red Hat パッケージ侵害の2日後、同系統のワームが「Phantom Gyp」と命名された新手法で複数メンテナーのアカウントから57パッケージを2時間以内に侵害した。npm の署名付きアテステーション（provenance attestation）はパイプラインそのものが乗っ取られた場合には保護効果がないことが実証された形で、サプライチェーンセキュリティの新たな課題として注目を集めている。ペイロードは4.2 MB の多層難読化ファイルで、クラウドプロバイダーのキー・CI/CD トークン・開発ツールの認証情報を一括収集する。
