CVE-2026-20245（Cisco Catalyst SD-WAN 0-day）— パッチ未提供・回避策なしで悪用継続、2026年7件目のSD-WAN脆弱性

🔗 Help Net Security

Cisco は6月5日、Catalyst SD-WAN Manager CLI に存在するコマンドインジェクション脆弱性（CVE-2026-20245）を公開した。netadmin 権限を持つ攻撃者が細工したファイルをアップロードすることで root として任意コマンドを実行できる。オンプレミス・クラウドプロ・クラウド管理・政府向け（FedRAMP）すべての SD-WAN 展開形態が対象で、現時点でパッチも回避策も提供されていない。Mandiant が悪用を報告しており、今年に入ってからの Cisco SD-WAN 0-day は本件で7件目となる。

security
🔗 NVD (NIST)

CISA は Linux Kernel のローカル特権昇格脆弱性 CVE-2026-31431 を Known Exploited Vulnerabilities（KEV）カタログに追加した。非特権のローカルユーザーが root 権限を取得できる欠陥で、主要 Linux ディストリビューションが影響を受ける。Red Hat を含む各ベンダーがパッチを提供しており、CISA の指定期限までに対象システムのアップデート適用が義務付けられている。
