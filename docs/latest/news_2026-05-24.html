Next Generation Events が developer preview に — フィールドレベルのトリガー制御・カスタムペイロード・変更追跡を実現

shopify
🔗 Shopify Developer Changelog

Shopify が Next Generation Events を developer preview として公開した（unstable API バージョン）。price フィールドを監視するサブスクリプションは title 変更時には発火しないなど、フィールドレベルのトリガーで不要な配信を排除できる。デリバリーペイロードは Admin GraphQL クエリで自由に定義でき、各配信に _changed フィールドが含まれるためどの変更がイベントを起こしたかを正確に追跡可能。現在は Product・Customer トピックが対応済みで、正式リリースへ向けて開発者のフィードバックを収集中。

Variant が Publishable に — API 2026-07 からバリアント単位でチャンネル別の可視性を独立制御

shopify
🔗 Shopify Developer Changelog

API バージョン 2026-07 から Product Variant が Publishable インターフェースを実装し、商品全体を非表示にすることなく特定バリアントをチャンネルや Catalog 単位で公開/非公開にできるようになった。デフォルトはすべてのバリアントが公開（オプトアウト型）で、既存アプリは変更なしで動作し続ける。チャンネルへのフィード対応アプリは Product Publicationsv2 でバリアント単位の公開状態を読み取る実装追加が推奨される。地域限定サイズや価格差異への対応が大幅に容易になる。

shopify
🔗 Shopify Developer Changelog

Shopify が App Automation Tokens を全アプリ向けに一般提供（GA）した（5 月 6 日付け）。Dev Dashboard から取得できるアプリスコープのトークンで、GitHub Workflows などの CI/CD システムから shopify app deploy を実行できる。Partner Dashboard の CLI トークンを置き換えるもので、トークンは 1・3・6 か月の有効期限付きかつローテーション機能内蔵でダウンタイムなく更新できる。既存 CLI トークンは失効まで動作するが早期移行が推奨される。

Shopify CLI の app release --force フラグが廃止予定 — 意図しない上書きリリース防止へ、移行対応が必要

shopify
🔗 Shopify Developer Changelog

Shopify CLI の shopify app release コマンドで使用できる --force フラグが非推奨となり、将来のバージョンで削除される予定が告知された。このフラグはドラフト状態のアプリをレビューなしに強制リリースするために使われていたが、意図しない上書きリリースを招くリスクがあった。CI/CD パイプラインや GitHub Actions で --force を使用しているチームは、App Automation Tokens と組み合わせた明示的なリリースフローへの移行を速やかに行う必要がある。

CISA KEV: CVE-2026-31431「Copy Fail」— Linux 特権昇格、130 以上の PoC 公開済み・連邦機関修正期限 5/15

security
🔗 The Hacker News

CISA が CVE-2026-31431（通称「Copy Fail」、CVSS 7.8）を Known Exploited Vulnerabilities カタログに追加した。Linux カーネルの特権昇格欠陥で、非特権の一般ユーザーが root 権限を取得できる。公開後数時間で 130 本以上の実証コード（PoC）が出回っており、現実の攻撃で既に悪用が確認されている。連邦機関の修正期限は 5 月 15 日だったが、一般企業でも多数の Linux ディストリビューションが影響を受けるため速やかなカーネルアップデートが必要。

CVE-2026-42945「NGINX Rift」— 2008 年以来のすべての nginx ビルドに存在するヒープバッファオーバーフロー、未認証 RCE

security
🔗 Security Boulevard

CVE-2026-42945（NGINX Rift）が 2008 年以来すべての nginx ビルドの ngx_http_rewrite_module に存在するヒープバッファオーバーフローであることが判明し、細工した HTTP リクエストを送るだけで未認証の遠隔コード実行が可能。CVE-2026-31431 と組み合わせることで、外部からの侵入→ root 昇格が連鎖的に成立するエクスプロイトチェーンが構築されている。nginx を使用するすべてのサーバーにパッチ適用が急務で、修正版の最新 nginx へのアップグレードが必要。
