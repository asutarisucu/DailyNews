【最終警告・残り2日】Shopify Scripts が 2026年6月30日に完全廃止 — 深夜を過ぎた瞬間にカスタムロジックが静かに停止

shopify
🔗 Shopify Developer Changelog

6月30日（2日後）の深夜をもって全Shopify Scriptsが完全停止する。4月15日以降すでに編集・公開が不可能になっており、6月30日終了後は未移行のカスタム割引・配送・支払いロジックがエラーメッセージなく静かに無効化される。移行先はShopify Functions（WebAssemblyモジュール）またはパブリックApp。Shopifyはこのデッドラインを二度と延長しないと明言しており、今日・明日中の移行完了確認が最後の機会。

Storefront MCP カートツールが UCP Cart MCP に移行 — get_cart/update_cart は 2026年8月31日廃止・PUT 全量更新セマンティクスに変更

shopify
🔗 Shopify Developer Changelog

6月24日付でStorefront MCP（https://{shop}.myshopify.com/api/mcp）のget_cart・update_cartが廃止予告された。代替は UCP準拠の Cart MCP（https://{shop-domain}/api/ucp/mcp）で、create_cart / get_cart / update_cart / cancel_cartが利用可能。重要な変更点として、update_cartはPUTセマンティクスを採用しており、差分ではなく毎回 line_items 配列を全量送信する必要がある。移行期限は2026年8月31日で、Spring '26 EditionのUCP（Universal Commerce Protocol）戦略への移行の一環。

🔗 Shopify Developer Changelog
