npm v12 が7月リリース予定 — allowScripts・allowGit・allowRemote がデフォルト無効化、CI/CD パイプラインの事前確認が必須

dev
🔗 GitHub Changelog

npm v12 が7月リリース予定で、サプライチェーンセキュリティを抜本的に強化する3つの破壊的変更が導入される。①allowScripts がデフォルト OFF（preinstall・install・postinstall スクリプトが実行ブロック）、②--allow-git がデフォルト none（Git 依存関係の解決が明示的許可制に）、③--allow-remote がデフォルト none（リモート URL 依存関係の解決が明示的許可制に）。npm 11.16.0 以降で事前警告が表示されるため、npm approve-scripts コマンドで許可スクリプトを確認し package.json を更新しておくことを推奨。
