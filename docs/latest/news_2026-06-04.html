App Events API 公開 — アプリのイベントを Dev Dashboard に集約、Shopify App Pricing の請求メーターとしても活用可能

shopify
🔗 Shopify Developer Changelog

App Events API を使ってアプリ内で発生するイベントを Shopify へ送信することで、Dev Dashboard の Logs セクションに Webhook・Function 実行・API コールと並べて一元表示できるようになりました。同じイベントを Shopify App Pricing の請求メーターとしてそのまま流用でき、利用量ベース課金（order_processed・email_sent など）の実装が大幅に簡略化されます。event_handle は App Pricing のメーターハンドルと完全一致（大文字小文字区別）が必要で、Dashboard の Logs フィルターで課金処理のリアルタイム確認も可能です。

🔗 Help Net Security
