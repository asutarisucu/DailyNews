Shopify CLI の --force フラグが廃止予定に — CI/CD パイプラインの対応が必要

shopify
🔗 shopify.dev changelog

Shopify CLI の app release --force フラグが廃止予定となりました。このフラグはエクステンションの削除を含む全確認プロンプトをスキップするため、誤ってアプリ拡張を削除してしまうリスクがありました。代わりに --allow-updates および --allow-deletes フラグを使用することで、CI/CDパイプラインの安全な無人実行が可能になります。対応が必要な開発者はスクリプトを更新してください。
